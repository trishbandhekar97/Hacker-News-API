# HackerNews API

Hacker News API developed with Node, Express and MongoDB

#### Create a .env file with the following values:

```
NODE_ENV=
PORT=
BASE_URL=https://hacker-news.firebaseio.com/v0
REDIS_PORT=
MONGO_HOST=
MONGO_PORT=
MONGO_DB_NAME=
MONGO_ROOT_USER=
MONGO_ROOT_PASSWORD=
```

### To run the API

```sh
$ docker-compose build
$ docker-compose up
```
